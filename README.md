# Analog Volume Meter

Simple microphone based volume meter with LM3915 and NE5532.

<details><summary>Component clarification</summary>
<p>

Q1: 2N3906 <br />
D1: 1N4148 <br />
U1: NE5532 <br />
U2: LM3915

</p>
</details>
